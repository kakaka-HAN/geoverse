# 运行和部署您的 AI Studio 应用

这包含了在本地运行您的应用所需的一切。

在 AI Studio 中查看您的应用：https://ai.studio/apps/63b9d377-7d6f-4e82-93b9-e34e5898293b

## 本地运行

**先决条件：** Node.js


1. 安装依赖项：
   `npm install`
2. 在 [.env.local](.env.local) 中设置 `GEMINI_API_KEY` 为您的 Gemini API 密钥
3. 运行应用：
   `npm run dev`
